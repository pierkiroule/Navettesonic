# Glossaire UI

Ce glossaire définit le vocabulaire produit autorisé pour l’interface.

## Termes autorisés

- **arène** : espace principal où se déroule l’activité.
- **bulle** : unité de contenu ou de conversation affichée dans l’arène.
- **publier** : action de rendre visible un contenu.
- **lien de partage** : URL permettant de partager l’accès à un contenu.

## Termes à retirer (legacy / ambigus)

Retirer et remplacer systématiquement les formulations suivantes dans l’UI et la documentation produit :

- `session live`
- `live session`
- `multiuser`
- `multi-user`
- `multi utilisateur`
- toute variante équivalente ambiguë

## Règle éditoriale

Utiliser exclusivement les termes autorisés ci-dessus dans les libellés, messages, aides contextuelles et textes marketing in-app.
